Features

为开发者打造的
全方位监控

从进程状态到 Token 消耗，从任务进度到子代理追踪，一目了然。

●

实时状态监测

自动发现所有运行中的 Claude Code 进程。区分 THINKING、EXECUTING、WAITING 三种工作状态，3 秒刷新，从不遗漏。

◆

Token 用量追踪

实时统计每个会话的 Input、Output、Cache Token 消耗。顶部汇总栏一眼掌握全局用量，详情面板精确到单进程。

■

子代理监控

追踪 Task 工具启动的子代理。显示每个子代理的类型、状态、Token 消耗、执行时长，支持折叠收起。

◇

任务进度条

基于 TodoWrite 的结构化任务列表，自动生成进度条和完成百分比。直观展示当前工作进度。

⚙

工具箱一览

查看当前 Claude Code 会话可用的全部 Skills 和 MCP 工具，双列布局，类型和说明清晰可见。

▼

日志流回溯

实时查看对话日志，支持展开查看 Edit diff、命令详情。按角色和类型标签分色显示，快速定位操作。

Getting Started

三步开始使用

零配置，一键启动。

01

安装依赖

确保已安装 Python 3.10+ 和 Node.js 18+。项目自带安装脚本，首次运行自动安装所有依赖。

02

一键启动

Windows 运行 start.bat，Linux/macOS 运行 ./start.sh。或直接执行 python run.py — 自动构建前端、启动服务、打开浏览器。

03

打开面板

浏览器访问 http://localhost:8765。启动 Claude Code 后，进程会自动出现在左侧列表中，点击即可查看详情。
